# Legacy Scripts

This directory contains the original Bash implementations of the Worktree Siblings tools.

## Contents

- `wt-clone.sh`: Prototype for `wts clone`
- `wt-new.sh`: Prototype for `wts new`
- `wt-done.sh`: Prototype for `wts done`
- `wt-common-lib.sh`: Shared bash functions

## Status

These scripts are **deprecated** and replaced by the TypeScript CLI (`wts`). They are preserved here for reference and historical context.
